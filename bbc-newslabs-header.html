Javascript API

The element supports traditional getters and setters. Try these in your javascript console.

const h=document.querySelectorAll('bbc-newslabs-header')[5];

// getter example - valid for app, applink, subtitle, userid, userinfo
h.app;

// setter example - valid for app, applink, subtitle, userid, userinfo
h.userid="bernim01";
h.userinfo="Miles Bernie";
h.subtitle="Documentation";
